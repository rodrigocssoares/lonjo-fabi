# Landing Page Lonjô + Andrea Orlandi + Fabi Guttin

Projeto em React + Vite + Tailwind.

## Como usar

1. Instale as dependências:
   npm install

2. Rode localmente:
   npm run dev

3. Gere build:
   npm run build

## Antes de publicar
Edite `src/App.jsx` e troque a URL do formulário:

const formUrl = "https://seuformulario.com/quero-participar";

## Publicação
Recomendado: Vercel
